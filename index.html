Calling All Station

In this project, my program simulates sending alert and cancel alert between multiple computers. When one computer receives an alert, it immediately sends it to another computer or computers, but receiving it assumes a delay. In addition, when the machine receives the command to cancel the alert, it immediately forwards it to other machines and stops sending the alert again.

Queen State

In this project, my program simulates the game called "queen". In this project, we can add and remove a queen, check the state of a queen, and check the number of survival queens.
